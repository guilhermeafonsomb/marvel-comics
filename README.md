# marvel-comics-api

I made this a front to marvel comics API. 

## Used tecnologies

1. TypeScript;
2. React
3. Chakra-UI;
4. Unit Test (RTL);
5. Vite;

## Setup Installation

1. Clone this repo;
2. Install ``` npm i ``` in your PC;

## Run application

1. To run this aplication ``` npm run dev ```;
2. To run tests ``` npm run test ``` ;
